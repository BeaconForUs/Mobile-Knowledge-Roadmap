# iOS开发 - 动画简介
---------------------
## 概述
动画是移动端开发比较重要的一部分。也许有些人因为公司业务的特点，日常工作中没什么机会涉及到动画，但是作为一个iOS开发者，掌握动画的基础知识仍然是必要的（至少能在面试中有所帮助吧😋）。
希望能在这篇文章中让大家对iOS动画有所了解，获得动画实现的一些思路，可以自己实现更加华丽的动画效果。
